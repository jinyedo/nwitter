## React App 생성시 오류 해결 방법

### 1. 일회성
```
npx create-react-app@latest 프로젝트명
```
### 2. 설치형 
```
npm install -g create-react-app@latest 
npx create-react-app 프로젝트명

-g OR --global
-v OR -version
```
